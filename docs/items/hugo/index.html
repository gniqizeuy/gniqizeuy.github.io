使用 yum 安装 hugo

新建 hugo.repo 文件, 访问 https://copr.fedorainfracloud.org/coprs/daftaupe/hugo/ 选择合适版本

vim /etc/yum.repos.d/hugo.repo

更新 yum

yum upgrade

安装 hugo

yum install hugo

hugo config

use inline HTML

add config like this

[markup]
  [markup.goldmark]
    [markup.goldmark.renderer]
      unsafe = true
