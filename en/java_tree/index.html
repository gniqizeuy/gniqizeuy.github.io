预备知识

树（tree）可以几种方式定义。定义树的一种自然的方式是递归。一棵树是一些节点的集合。这个集合可以是空集；若不是空集，则树有称作根（root）的节点 r 以及 0 个或多个非空的（子）树 T1, T2, … Tk 组成，这些子树中每一棵树都被来自根 r 的一条有向的边（edge）所连结。

每一棵子树的根叫做根 r 的 child，而 r 是每一棵子树的根的 parent。下图是示用递归定义的典型的树：
